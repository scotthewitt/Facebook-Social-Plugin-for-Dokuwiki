About

A plugin for Dokuwiki which offers easy to use inclusion of Facebook like and comment social plugins.

Syntax

<fbl> ~ Inserts a Facebook like button

<fbc> ~ Inserts a Facebook comment box this also includes a like button.

Implementation

This is my first dokuwiki plugin and as such I imagine is not done the best way. This syntax plugin substitutes the code lines above for javascript encapsulated with html tags.
